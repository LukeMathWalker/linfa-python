# Linfa

Python bindings for the Rust crate [`linfa`](https://github.com/LukeMathWalker/linfa).

## Requirements

* Rust nightly compiler (`2019-10-17` or later)
* Python 3.7+
* [`poetry`](https://github.com/sdispater/poetry)

## Build

```bash
poetry install
```

## Tests 

```bash
poetry run pytest
```
